VertoTask Project - Source Code


1. Open the solution file `VertoTask.sln` in Visual Studio 2022 or newer
2. Restore NuGet packages (usually automatic)
3. Build the solution in Debug
4. Run the project -  localhost automatically

The database connection string is in `appsettings.json`
Images and static files are in `wwwroot/images`
For the admin page, use `/Admin/EditHomepageText`

For the database backup, it is in 'DatabaseBackup'
